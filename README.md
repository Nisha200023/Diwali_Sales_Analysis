# Diwali_Sales_Analysis
🎇 Diwali Sales Analysis
📌 Project Overview

This project analyzes Diwali sales data to identify customer purchasing behavior, sales trends, and business insights. The analysis helps understand which customer segments contribute most to sales and provides data-driven recommendations for improving marketing and sales strategies.

🎯 Objectives
Analyze customer demographics (Age, Gender, Marital Status, Occupation, State).
Identify top-performing products and categories.
Understand customer purchasing patterns during the Diwali season.
Generate actionable business insights using data visualization.
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

📊 Analysis Performed
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Customer Segmentation
Sales Trend Analysis
Product Category Analysis
Data Visualization

🔍 Key Insights
Married women aged 26–35 years contributed significantly to overall sales.
Customers from specific states showed higher purchasing power.
Certain product categories generated the highest revenue during the Diwali season.
Targeted marketing can improve sales performance.
📁 Dataset

The dataset contains customer demographics, product details, order information, and purchase amounts collected during the Diwali sales season.

🚀 Conclusion

The project demonstrates how data analytics can be used to uncover customer behavior and sales trends, enabling businesses to make informed decisions and improve profitability during festive seasons.
